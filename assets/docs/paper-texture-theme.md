# Paper Texture Manager Theme Prompt

This prompt describes a paper-like UI theme with a fine-grain texture and soft modern tone suited for an admin dashboard.

## Prompt (JP)
管理画面UIのデザイン。紙のような微細テクスチャを背景に使用したテーマ。色味は白〜淡いグレーのみで黄ばみなし。境界線は鉛筆のように薄く柔らかい。ボタンやパネルは白ベースで角を控えめに丸める。全体的に「クラフト紙×モダンUI」の軽いニュアンス。左ペインの白背景とも自然に調和し、ノスタルジックに寄りすぎない落ち着いた印象。紙のような微細テクスチャ、柔らかくクラフト感のあるUI。

## Design Notes
- Use only white to light-gray hues; avoid yellow tinting.
- Keep paper texture extremely fine and subtle; do not use coarse grain.
- Lines and separators should be thin and low-contrast, avoiding heavy black strokes.
- Corners are softly rounded; controls remain modern and minimal.
- Maintain a light craft-paper mood without leaning into nostalgic styling.
