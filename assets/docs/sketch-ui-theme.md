# Sketch UI（スケッチUI）テーマ プロンプト

## 概要

管理画面の構造やコンポーネントはそのままに、線だけを手描きスケッチ風へ差し替えるデザイン指示です。背景は白〜クリーム色に限定し、漫画風や太線を避けて親しみやすいラフ感を持たせます。

## デザイン原則

- 線は極細の手描き鉛筆線のように波打たせる。太線や漫画的なトーンは禁止。
- 背景は白〜淡いクリーム色に限定し、カラフルな配色は使用しない。
- UIレイアウト・構造は通常の管理画面と同じ。枠線のみラフな手描き風アウトラインにする。
- ボタンやパネルにも軽い手描きアウトラインを付与しつつ、形や配置は従来のまま。
- アイコンや文字は通常描画を維持し、手描き加工を加えない。
- 全体として「手描き風の線、軽いラフ感、イラスト調」の雰囲気を目指す。

### コンポーネント別の描写ヒント

- サイドバー / ヘッダー: 枠線は揺らぎのある鉛筆線。背景は白〜クリームのフラット塗りで陰影なし。
- テーブル: 罫線を極細の手描き線に置き換え、行の区切り線にわずかな途切れや揺らぎを加える。
- フォーム要素: 入力枠のアウトラインをラフに。選択状態は淡いハイライトで、塗りは使わず線の質感で差を付ける。
- ボタン: 角丸は最小限に抑え、外枠のみ手描き線。ホバーは背景色を変えず、線の濃度を少し上げる程度。
- パネル / カード: 四辺の線を均一でなく手描き風に。シャドウは入れず、余白で段差を表現。

 ## 生成用プロンプト例

 > 管理画面UIのデザイン。手描きスケッチ風のラインを使ったテーマ。境界線は極薄でラフな鉛筆線のように波打つ。背景は白または淡いクリーム色でシンプル。ボタンやパネルも軽く手描き風アウトラインを付与。文字やアイコンは通常のまま、UI全体が軽いイラスト調の雰囲気。親しみやすく、遊び心のあるデザイン。

### そのまま使えるマスタープロンプト

> 管理画面テーマ。UI構造は通常の管理画面を踏襲し、線だけを極細の手描き鉛筆線に置き換える。背景は白〜淡いクリーム色でベタ塗り、カラフルな配色は禁止。サイドバーとヘッダーは揺らぎのある線で囲み、テーブル罫線は細くラフに、行の区切りにわずかな途切れを入れる。フォーム入力枠とボタンには軽い手描きアウトラインを付け、ホバー時も線の濃淡で表情を付けるのみ。パネルやカードは影なしで、ラフな四辺の線と十分な余白で区切る。アイコンと文字は通常描画を保ち、漫画風や太い線は禁止。全体を「手描き風の線、軽いラフ感、イラスト調」でまとめる。

## 補足ヒント

- 配色は背景の明度差で軽く階調を付ける程度にとどめ、彩度を上げない。
- 手描き線は部分的に途切れさせたり揺らぎを出すと自然さが増す。
- 影やグラデーションは極力抑え、シンプルな塗りと線の質感でまとめる。
- 「通常の管理画面と同じUI構造」「線のみ手描き化」「文字とアイコンは加工しない」を必ずセットで指定する。
