# セミガラス管理画面テーマ プロンプト

柔らかい白基調のセミガラス調 UI を生成するための指示セットです。ガラス効果を控えめにしつつ、実用性と可読性を保った管理画面デザインを得ることを目的としています。

## デザイン要件
- 背景は常に白基調。暗色ベースや濃い色のグラデーションは禁止。
- フロストガラス効果はごく薄く、過度な透明感や透過を避ける。
- 光沢表現は最小限にし、ハイライトは柔らかいライティングでごく控えめにする。
- パネルやヘッダーには淡いグラデーションを使い、境界線は極細で透明感を引き立てる。
- 影はソフトで拡散したものにとどめ、要素の立体感をさりげなく演出する。
- ツリーペインやフォーム背景は白を基調にし、周囲と自然に溶け込む。
- 全体としてミニマルで清潔感のある管理画面らしいトーンを維持する。

## 生成用プロンプト
管理画面UIのデザイン。白を基調にした半透明ガラス風テーマ。背景に薄いフロストガラス効果。パネルやヘッダーには淡いグラデーションとソフトな光沢。境界線は極細で、透明感を引き立てる。影は柔らかく拡散。ミニマルで清潔感のあるデザイン。ツリーペインは白背景で自然に溶け込む。全体的に透明感のある白系ガラス風で、光沢はごく薄く、暗色背景は避ける。
