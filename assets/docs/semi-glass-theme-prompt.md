# セミガラス管理画面テーマ プロンプト

柔らかい白基調のセミガラス調 UI を生成するための指示セットです。ガラス効果を控えめにしつつ、実用性と可読性を保った管理画面デザインを得ることを目的としています。

## デザイン要件
- 背景は常に白基調。暗色ベースや濃い色のグラデーションは禁止。
- フロストガラス効果はごく薄く、過度な透明感や透過を避ける。
- 光沢表現は最小限にし、ハイライトは柔らかいライティングでごく控えめにする。
- パネルやヘッダーには淡いグラデーションを使い、境界線は極細で透明感を引き立てる。
- 影はソフトで拡散したものにとどめ、要素の立体感をさりげなく演出する。
- ツリーペインやフォーム背景は白を基調にし、周囲と自然に溶け込む。
- 全体としてミニマルで清潔感のある管理画面らしいトーンを維持する。

## カラートーンと素材感
- ベース: #ffffff〜#f7f8fb の白系グラデーションをメインに使用。
- 枠線/仕切り: #e7e9ef 付近の極細線。透明度 70〜80% 程度で存在感を抑制。
- アクセント: ごく淡いミント系 (#e8f3f7) またはウォームグレー (#f4f1ed) をポイントで使用。
- ガラス感: ほんのりフロスト（5〜8px のブラー相当）。高い透過率やギラつきは禁止。
- 光沢: 角を丸め、ソフトなハイライトをうっすら乗せる程度。鏡面反射は避ける。

## レイアウト指針
- ヘッダー: 白ベースに薄いグラデーション、アイコンはシンプルで低コントラスト。
- サイドナビ/ツリーペイン: 完全な白背景。ホバー時のみごく淡いシャドウかアウトラインを付与。
- カード/パネル: 角丸 + 薄いフロスト。タイトルは 14–16px、本文は 13–14px の明瞭なサンセリフ。
- フォーム: 入力枠は細いアウトライン、フォーカス時のみ淡いグロー。エラーカラーも彩度を抑える。
- 影: ぼかしの強いドロップシャドウを低不透明度で使用し、浮きすぎない深さに留める。

## 生成用プロンプト（詳細版）
管理画面UIのデザイン。白を基調にした半透明ガラス風テーマ。背景に薄いフロストガラス効果。パネルやヘッダーには淡いグラデーションとソフトな光沢。境界線は極細で透明感を引き立てる。影は柔らかく拡散。ツリーペインとフォームは白背景で自然に溶け込む。文字は高可読性のサンセリフ、コントラストは控えめ。全体的に透明感のある白系ガラス風で、光沢はごく薄く、暗色背景や強い彩度は避ける。

## 生成用プロンプト（クイック版）
白基調の管理画面UI、セミガラス調。薄いフロストガラス効果、淡いグラデーションとごく薄い光沢。極細の境界線、ソフトで拡散した影。ツリーペインとフォームは白背景で一体化。ミニマルで清潔感、暗色背景と強い透明感は禁止。

## 避けるべき表現（ネガティブプロンプト例）
- 暗い背景、黒〜濃紺ベースの配色
- 強いガラス反射、鏡面仕上げ、過度な透明度
- 原色や高彩度のアクセント、蛍光色
- 厚い枠線、硬いドロップシャドウ、強いグロー
- メタリック質感、クローム質感
